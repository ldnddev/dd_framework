Slider Component Breakdown

Overview

The dd-slider component provides a responsive carousel for images and copy with previous and next buttons as wells and dot navigation.

This component is ideal for when space is limited but there are several items to be displayed. Mostly used for images with possible copy block for descriptions or CTA functionality.

Basic Usage

Wrap slide items in a div with class dd-slider.

Accessibility and Best Practices

Ensure backgrounds provide sufficient contrast. Nest semantic elements inside. Test landmark navigation with screen readers. In CMS, make titles optional but encouraged.
