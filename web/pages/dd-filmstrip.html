Filmstrip Component Breakdown

Overview

The dd-filmstrip component provides a marque like section that has an infinite scroll with a pause effect on hover.

This component is ideal for showing logos or small thumbnail images that continually scroll.

Basic Usage

Wrap the Filmstrip in a div with class dd-filmstrip. Use an unordered list to show images.

CSS Modifiers (from _dd_filmstrip.scss)

The SCSS file defines base styles for the Filmstrip, animation direction. Key modifiers for .dd-filmstrip include:

  • -reverset: reverse the play head.

Accessibility and Best Practices

Ensure overlay text contrasts with background (use borders/opacity). Provide alt text for images if visible. In CMS, validate background images for required fields and enforce contrast checks if possible.
