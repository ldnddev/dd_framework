# Task
This is your engagement system - designed for clarity, energy, and ADA-compliant interaction states. Provide light and dark mode colors for each section listed under "Variables" based on the "Base colors".
- Use the "Base colors: Primary, Seconday and Tertiary" colors to create proper variations for the "Primary, Seconday, and Tertiery" sections listed under the "Variables" section for intuitive feedback across light and dark UIs
- Use the "Base colors: Primary, Seconday and Tertiary" colors to create proper variations for the "Action (CTAs / Buttons / Interactions)" section listed under the "Variables" section for intuitive feedback across light and dark UIs
  - Each state consists of default, hover, pressed, and disabled with coordinated surface, border, and text colors
- Use the "Base colors: Support / Utility" color to create proper variations for the "Support / Utility" section listed under "Variables" for intuitive feedback across light and dark UIs
- Be sure that there is enough contrast between all the new "Variables" and the "Text Colors" to pass WCAG compliance
- Be sure to provide an easy copy and paste section of the variables
- Be sure to provide quick WCAG contrast notes on how to best use each new section of colors
